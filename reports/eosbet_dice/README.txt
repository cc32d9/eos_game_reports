EOSBet Dice fairness reports
============================


Commands executed:

```
perl scripts/eosbet_dice_getdata.pl

perl scripts/eosbet_dice_report.pl \
 --out=reports/eosbet_dice/eosbet_dice_fairness_report_2018-09-06.xlsx \
 --end=2018-09-06 

```

The PDF files are generated by Microsoft Excel, by setting up page
layout and saving as PDF. The "Winners and Losers" worksheet is omitted
from PDF exports because of its size.
